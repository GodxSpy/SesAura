# SesAura
Final year Project 
A smart acoustic alert system for assisting hearing-impaired individuals
Implements real-time sound detection and classification using Raspberry Pi.
Detects critical household sounds:door knock, baby crying, fire alarm, glass break,dog bark and screams.
Provides multi-modal alerts: mobile notifications (with emoji + vibration) and LED indicators.
Integrates speech-to-text for converting detected speech into readable text.
Lightweight, edge-deployable, and designed for safe and independent living.
